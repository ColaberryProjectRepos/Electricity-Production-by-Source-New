Renewable and Non Renewable Sub KPIs

1. Click card visualizations twice, each for ‘Renewable YOY’ and 'NonRenewable YOY’ from 'MeasuresForElectricity'  to values field respectively.

2. For Renewable YOY card, In the format->visual, in callout value, set font style =Din, size =15, color= #6D5A00, category label= off

3. In the format->general, set

Height=33, Width=68, Horizontal=55, Vertical=270, Title=off, Effects->background=off

4. For NonRenewable YOY card, In the format->visual, in callout value, set font style =Din, size =15, color= #A38600, category label= off

5. In the format->general, set

Height=33, Width=68, Horizontal=166, Vertical=270, Title=off, Effects->background=off

6.Format = percentage, decimal =1
