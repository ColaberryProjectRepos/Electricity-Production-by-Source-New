Quest toward Renewable (Waterfall Chart) 22


1.    Before going to waterfall chart, create a measure description table Create the following table  

-          On the Power BI Desktop Home ribbon, Click on Enter data, name the        first column header as ID and the second column header as Sources and Name the table “.RenVsNonRen”

 

1.    Next, Create Measure

Variance = SWITCH(SELECTEDVALUE('.RenVsNonRen'[ID]),1,[Hydro],2,[Coal]*-1,3,[Wind],4,[Gas]*-1,5,  [OtherRen],6,[Nuclear]*-1,7,[Solar],8,[Oil]*-1)

               Format :- Percentage decimal =1

2.    Now, Create Waterfall Chart using the below steps

1. Click on an empty space on the canvas, and from the visualizations pane select Waterfall Chart.

2. Drag Sources for x axis From .RenVsNonRen Table and Variance for Column y-axis from electricity measure.

3. Drag all the Tooltips = RenProd, Renewable YOY, NonRenProd, Non Renewable YOY, RenVsNonRenRatio,Hydro prod, Hydro, Coal prod, Coal, Wind Prod, Wind, Gas prod`, Gas, OtherRen Prod, OtherRen, Nuclera Prod, Nuclear, Solar Prod, Solar, Oil Prod, Oil

In the Formatting Tab->

        X-axis: On, Font =Arial, size = 12, Color = #1A1A1A,Title = Off

3.    Y-axis: On, Font =Arial, size = 12, Color = #1A1A1A. Title: Off.

4.    Legend: On, Position = Top left,Text: Font = Calibri, size =12, Color =#1A1A1A. Title: Off.

5.    Gridlines: Horizontal: Off.

6.    Column Color = Increase #1AAB40, Decrease #F82A1B, Show total column = Off.

7.    Data labels: Off

8.    5. In the Formatting Tab->

9.    General->Properties->Size: Height = 311, Width = 345, Position: Horizontal = 228, Vertical =405,

10.    Title: Quest towards Renewable, Font = DIN, Size = 16, Color = #FFFFFF, Background= #A38600 , Center alignment

11.    Effect: Background Color= #A0D1FF, Trasparency = 70%.

12. Visual Border =#808080

13. Tooltips: On, Font =Din, Size = 10, Label Color = #C8C8C8, Value Color #EAEAEA.
