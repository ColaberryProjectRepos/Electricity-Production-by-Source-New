3. Rename the table Electricity Production by Source to ' Electricity '



4. Now to change ‘year’ column to date format, first Duplicate the Year Column and rename it to Year Source,

5. Then change Original year column data type to text. To do that, in the home ribbon, under transform section there is datatype button and click it and select text.

6. Sort the year column to ascending again (by clicking arrow next to column name and select ascending) , now in the transform ribbon click datatype and select ‘date’.

7. Now the year is in date format, change that column name to ‘Date’.

8. Right Click on the Date Column and Transform-> Year->End of the Year.

9.Now the data in this table is ready to load, click on file and select close & apply. You will see that table loaded into power BI.


10. Repeat the steps 2-4 from (Step 1 Importing Data) to import Country Continent dataset Excel file.

11. From power query editor, select Country, continent and subregion columns by pressing Ctrl key and Right click and click Remove Other Columns



12. Select Merge Queries from Home in the ribbon and Choose Continent Country table and Electricity Table and select Country columns from both the tables and Choose Left Outer Join. Click Ok.

13. Electricity Table will be added with a new column-> Expand that column by clicking on Expand button and Select Continent and Country to Expand and hit ok.

14. Rename Both the columns as Country_ and Continent. Right click each column and unselect Null values from both the columns. Then Click Remove Empty from Continent column.


 

15. Add 2 custom columns for Total Electricity Production by Source using the below formula and data type as Decimal Number.

Renewable_Energy_Production =    Solar + Wind + Other Renewable + Hydro

NonRenewable_Energy_Production =   Oil + Gas + Nuclear + Coal
