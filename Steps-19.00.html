KPI Indicators 19

RenVsNonRen Indicators

1. Click on the card visualizations,  drag 'RenVsNonRen indicator’ for card from 'Indicater Measures' to values fields respectively.

2. For 'RenVsNonRen’ indicator' card, In the format->visual, in callout value, set font size 28, color -> click fx(conditional formatting) set format style= rules, field based on = 'RenVsNonRen'

value >=0 and <max then #078405(Green)

value >=min and <0 then #F82A1B (Red)

category label= off

3. In the format->general, set

Height=45, Width=47, Horizontal=182, Vertical=123, Title=off, Effects-background=off
